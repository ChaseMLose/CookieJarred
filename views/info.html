Info:

Cookie Jarred is an open source extension that is actively being updated.

Cookie Jarred runs every time a new website is loaded. It will search for the cookie confirmation layer if by HTML class and will attempt to remove it. If the cookie layer does not get removed pleas submit a Github issue via the "Report a Bug" button on the main page. When reporting a cookie please include the link of the website where it occurred.
